# easier-minecraft-fabric-1.21

Easier Minecraft mod for fabric 1.21~1.21.1

### Overview
- Add overpowered bows. 
- Add overpowered food. 
- Add overpowered enchantments. 
- Add recipe for totem-of-undying. 
- Add recipes for furnace / blast furnace so that raw ore blocks can be converted to ore blocks directly. 
- Add large veins for diamond / ancient debris. 
- Overwrite enchantment exclusive sets so that Mending and Infinity, Multishot and Piercing are compatible. 
- Overwrite vanilla Infinity so that tipped-arrow and spectral-arrow can be shot unlimitedly. 
- Enhance vanilla Sharpness / Protection / Power / Punch enchantments. 
- Multishot and Piercing can be used on bows. 
- Pressing jump during elytra flight will stop flying. 
- Librarians only sell enchantment books of the highest level. 

### Overpowered bows
- Explosion Bow: Create non-destructive / destructive explosion when used in main hand / off hand, respectively.  
- Teleport Bow: Teleport. 
- Firework Rocket Bow: A cheaper substitude for firework rocket. 
- Villager Bow: Spawn a villager. 

### Overpowered food
- Emerald Apple: Apply powerful effects. 

### Overpowered enchantments
- Psychedelic: Apply debuff to the enemy. 
- Sonic Guard: Reduce damage caused by sonic boom. 
- Experience Harvest: Provide bonus experience and heal the attacker after attacking.
